    OpenICE on FHIR

    A prototype application for viewing FHIR exported OpenICE data

    In an effort to enable key data integration functionality, OpenICE has prototyped a medical device data export app with HL7 FHIR. This functionality enables clinical applications like EMR flowsheets to recieve OpenICE data from a standard FHIR interface. This web application retrieves and visualizes patient and observation resource data stored on the HAPI FHIR valadation server at fhir.openice.info. The data in this server was exported by an OpenICE Supervisor running the HL7 Exporter app.
